# Hướng dẫn TTS (Text-to-Speech)

My Translator hỗ trợ 2 chế độ đọc bản dịch thành tiếng:

## Edge TTS ⭐ (Mặc định — Miễn phí)

Giọng đọc tự nhiên từ Microsoft, không cần API key.

**Giọng có sẵn:** HoaiMy 🇻🇳 (nữ), NamMinh 🇻🇳 (nam), Jenny 🇺🇸, Guy 🇺🇸, Nanami 🇯🇵, SunHi 🇰🇷, Xiaoxiao 🇨🇳

**Tốc độ:** chỉnh từ -50% đến +100% trong Settings → TTS. Mặc định +50%.

> Cần kết nối internet. Không cần tài khoản.

## ElevenLabs (Trả phí)

Giọng AI cao cấp, cần API key từ [elevenlabs.io](https://elevenlabs.io).

Chọn trong Settings → TTS → Provider → ElevenLabs → dán API key.

## Xử lý sự cố

- **Không có tiếng?** Kiểm tra nút 🔊 đã bật, và âm lượng hệ thống.
- **Edge TTS không hoạt động?** Kiểm tra kết nối internet.
