# TTS (Text-to-Speech) Guide

My Translator supports 2 TTS providers:

## Edge TTS ⭐ (Default — Free)

Natural voices from Microsoft, no API key needed.

**Voices:** HoaiMy 🇻🇳 (F), NamMinh 🇻🇳 (M), Jenny 🇺🇸, Guy 🇺🇸, Nanami 🇯🇵, SunHi 🇰🇷, Xiaoxiao 🇨🇳

**Speed:** adjustable from -50% to +100% in Settings → TTS. Default +50%.

> Requires internet. No account needed.

## ElevenLabs (Paid)

Premium AI voices, requires API key from [elevenlabs.io](https://elevenlabs.io).

Select in Settings → TTS → Provider → ElevenLabs → paste your API key.

## Troubleshooting

- **No sound?** Check 🔊 button is active and system volume.
- **Edge TTS not working?** Check internet connection.
